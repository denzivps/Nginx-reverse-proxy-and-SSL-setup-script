# theme installer

This automated script allows you to setup your domain effortlessly in just a few seconds.

## How to Use
To setup, simply run the following command:

```bash
curl -sSL https://raw.githubusercontent.com/denzivps/Nginx-reverse-proxy-and-SSL-setup-script/refs/heads/main/setup-proxy.sh > setup-proxy.sh && chmod +x setup-proxy.sh && sudo ./setup-proxy.sh
```
